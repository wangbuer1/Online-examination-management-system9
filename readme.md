## 2、项目技术

后端框架：Spring、SpringMVC、Hibernate

前端框架：Bootstrap、jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven版本：无限制
- 硬件环境：Windows 或者 Mac OS
